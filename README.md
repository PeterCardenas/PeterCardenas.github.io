# Portfolio Website

This is my portfolio website made using ReactJS and HTML/CSS.

## Install

`npm install`

## Personalize

- Change `src/resume_data.js` to match your information
- Go to skill bars section in `public/css/layout.css` to change skill level
- Change `public/images/profilepic.jpg` to your profile picture
- Change `public/images/header-background.jpg` to your background picture
- Change `public/images/favicon.ico` to your logo
- Change everything in `public/images/portfolio/` to your project pictures and change image urls in portfolio section of `src/resume_data.js` accordingly
- Change `public/images/Peter_Cardenas_Resume.pdf` to your own resume and change its link in `src/resume_data.js`

## Run

### Local

`npm start`

### Deploy

- Change `homepage` in `package.json` to be your website URL
- `npm run deploy -- "<commit message>"`
